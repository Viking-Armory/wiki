**## install epel RH8**\\
<code>
sudo subscription-manager repos --enable codeready-builder-for-rhel-8-$(arch)-rpms\\
sudo dnf install https://dl.fedoraproject.org/pub/epel/epel-release-latest-8.noarch.rpm\\
</code>
